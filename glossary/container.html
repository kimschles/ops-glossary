[WIP] Container

_Short and Sweet_ A container is a way of 'packaging code' and its dependencies so that an application can easily be run on different computing enviornments. _More Info_ Containers vs. VMs Old way: one server per application VM Way: Code, dependencies _and_ the operating system are packaged togethe New Way: just the code and dependencies
